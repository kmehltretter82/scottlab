ScottLab for instructors

ScottLab is a free, open-source (Apache-2.0), browser-based companion for teaching Scott information systems. Every lesson runs on a deterministic, tested semantic core; every visual action has a precise mathematical meaning; everything works with keyboard, mouse, or touch, respects reduced motion, and is available in English and German.

Shareable links

Every lesson and gallery system has a stable hash URL, and sandbox selections are encoded in the link — for example #/sandbox/flat-boolean/delta.true opens the sandbox with the state {Δ, true} selected. Lesson progress persists in the student's browser, so lessons are resumable across sessions.

Embedding a widget in lecture notes

A self-contained state-order explorer can be embedded in any page via an iframe. Choose the system with ?system= and the language with &lang=de: 

<iframe
  src="https://kmehltretter82.github.io/scottlab/embed.html?system=coquand"
  width="640" height="320" loading="lazy"
  title="ScottLab: the Coquand system"></iframe>

Available system ids: flat-boolean, coquand, access-permissions, editing-policy, bounded-lazy-naturals, stream-prefixes, take-away-game.

Bring your own systems

The gallery imports versioned JSON documents (validated structurally and against the information-system laws before activation) and can derive a system from a small formal-concept cross-table. Exports are deterministic, so example files diff cleanly in version control — convenient for problem sets.
